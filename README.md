 # Installation

````
git clone https://github.com/ZonaAirdrop/Pharos-Bot.git
````
````
cd Pharos-Bot
````

# 1.Install Dependencies
````
pip install -r requirements.txt
````
# accounts.txt `nano accounts.txt`
````
your_private_key_1

````

(Optional) Create `proxies.txt`:

Each proxy must be on a new line:

```

```
